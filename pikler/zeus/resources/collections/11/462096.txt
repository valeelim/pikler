Query language     a query language is a formal language that is used to make queries to database systems , and other information systems . the result of a query is a subset of the data . there are different query languages which target different user groups . it is possible to translate a query from one query language into another . - xquery is a query language to search xml information systems - xpath is used to navigate through xml documents - sql is a query language commonly used for database systems - query by example was developed so that users who rarely use databases can also formulate queries .
