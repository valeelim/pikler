Jason Upton     jason upton ( born december 15 , 1973 ) is an independent christian worship leader and musician . he has a large underground following . he has released 12 albums since the year 2000 : - `` key of david '' ( 2000 ) - `` faith '' ( 2001 ) - `` jacob 's dream '' ( 2002 ) - `` dying star '' ( 2002 ) - `` remember '' ( 2003 ) - `` trusting the angels '' ( 2004 ) - `` great river road '' ( 2005 ) - `` open up the earth '' ( 2005 ) - `` beautiful people '' ( 2007 ) - `` between earth and sky '' ( 2007 ) - `` 1200 ft below sea level '' ( 2008 ) - `` on the rim of the visible world '' ( 2009 ) w/ the good land band
