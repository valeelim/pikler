Anaconda (installer)    anaconda ( installer ) anaconda is the installer for red hat linux and fedora . it is written in python and c , with a graphical interface using gtk+ 3 and pygobject and a text inetrface using python-newt . a kickstart file can be used to automatically configure the installation , allowing users to run it without much supervision . anaconda is used by rhel , fedora and a number of other projects , anaconda offers a text-mode and gui mode , so users can install on a wide range of systems . anacondas are lizard-eating boas , and the caldera installation program was called `` lizard '' , hence this name was chosen . other websites . - anaconda home page
