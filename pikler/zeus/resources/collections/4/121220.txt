Live USB system creator     the live usb system creator is to tool designed to create live usbs of ubuntu from a running ubuntu live cd features . - detects available usb flash drives ( using hal ) - partitions usb flash drive with 1 partition - sets partition bootable - writes mbr to usb flash drive - formats partition fat32 - installs bootloader ( syslinux ) to partition - writes bootloader configuration file - copies necessary files from running live cd to usb flash drive - sets language and keyboard of usb live system to match running live cd - optionally : downloads and integrates adobe flash player - optionally : enables persistent /home ( ubuntu 8.10 and later only ) other pages . - list of tools to create live usb systems - ubuntu live usb creator
