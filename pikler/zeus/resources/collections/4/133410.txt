Ubuntu Live USB creator     ubuntu live usb creator is an official tool to create live usbs of ubuntu from an iso image . the tool is already included in the repositories of ubuntu 8.10. features . - optional persistence ( all changes made to the system are saved , unlike a live cd ) - progress indicator - does not format the device other pages . - list of tools to create live usb systems other websites . - project page at launchpad - cd2usb - similar tool ( for microsoft windows )
