Fedora Live USB creator     the fedora live usb creator is an official tool designed to create live usbs of fedora . features . - cross-platform ( available for windows and linux ) - non-destructive install ( does not format the device ) - supports various fedora releases , including fedora 19 - automatically detects all removable devices - persistent storage creation , to save all documents created and modifications made to the system ( only works with fedora 9 ) - sha1 checksum verification of known releases , to ensure that there is no corruption when downloading related pages . - list of tools to create live usb systems
