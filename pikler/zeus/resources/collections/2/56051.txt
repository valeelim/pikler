MySQL     mysql is a database system used by many websites on the internet . it is based off of sql . many ways of doing things in sql are similar in mysql . mysql was first released in may 1995 and a windows version was released in january 1998. the latest version ( 5.6.11 ) was released in april 2013. mysql is compatible with many major programming languages . the most common one is php . an odbc interface ( called myodbc ) has been made so that users of microsoft 's asp language can use mysql . some of the largest mysql users on the internet include : - apple - amazon.com - digg - google - baradbabaei many php scripts also use mysql . these include : - wordpress - phpbb - mediawiki - the software behind wikipedia related pages . - php other websites . - database master - mysql management studio
