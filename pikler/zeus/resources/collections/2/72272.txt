Long distance race     a long distance race is any running event on the track that is longer than 3000m . this includes such events as the 3000m run , the 3000m steeplechase , the 3200m run , the two mile run , the three mile run , 5000m run , five mile run , six mile run , and 10000m run , and so on . related pages . - sprint - middle distance race - walking race - hurdle race - relay - steeplechase ( athletics ) - cross-country race - marathon
