KOffice     koffice is a free office suite for the k desktop environment ( kde ) . all its components have been released under free software/open source licenses . the latest version of koffice is 1.6.3 , which was released on june 7 , 2007. koffice is primarily designed for unix operating systems , but in the future there will likely be a microsoft windows and native mac os x version of all the applications . koffice applications uses opendocument as their native file format when possible . the suite is released separately from kde and can be downloaded at the koffice homepage . koffice includes the following components : koffice includes import filters for some file formats provided by competitors . other websites . - koffice website
