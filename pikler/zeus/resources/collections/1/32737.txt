Cottbus     cottbus is a city in the east of germany , in the state of brandenburg . cottbus has about 105,000 inhabitants and is the second largest city after potsdam in brandenburg . it lies southeast of berlin on the river spree . there are two universities , the brandenburgische technische universität ( btu ) and the hochschule lausitz . boroughs of cottbus . - ( 1 ) mitte ( srjejź ) - ( 2 ) neu und alt schmellwitz - ( 3 ) sandow ( žandow ) - ( 4 ) spremberger vorstadt - ( 5 ) ströbitz ( strobice ) - ( 6 ) sielow ( žylow ) - ( 7 ) saspow ( zaspy ) - ( 8 ) merzdorf ( žylowk ) - ( 9 ) dissenchen ( dešank ) - ( 10 ) branitz ( rogeńc ) - ( 11 ) madlow ( modłej ) - ( 12 ) sachsendorf ( knorawa ) - ( 13 ) döbbrick ( depsk ) - ( 14 ) skadow ( škodow ) - ( 15 ) willmersdorf ( rogozno ) - ( 16 ) kahren ( korjeń ) - ( 17 ) kiekebusch ( kibuš ) - ( 18 ) gallinchen ( gołynk ) - ( 19 ) groß gaglow ( gogolow )
