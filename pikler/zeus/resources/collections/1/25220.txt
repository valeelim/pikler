Nationalities of China     there are many people in china , more than in any other country . they all have many different cultures , histories and beliefs . the chinese government officially says there are 56 nationalities . they are called `` zú '' or `` mínzú '' in china . of these , almost all of them ( about 90 % ) are han people ( 汉族 : hàn zú ) . the rest of these nationalities ( called `` ethnic minorities '' ) , from biggest to smallest in population , are : - zhuang ( 壮族 : zhuàng zú ) - manchu ( 满族 : mǎn zú ) - hui ( 回族 : huí zú ) - miao ( 苗族 : miáo zú ) ( hmong ) - uyghur ( 维吾尔族 : wéiwúěr zú ) - yi ( 彝族 : yí zú ) - tujia ( 土家族 : tǔjiā zú ) - mongol ( 蒙古族 : měnggǔ zú ) - tibetan ( 藏族 : zàng zú ) - buyei ( 布依族 : bùyī zú ) - dong ( 侗族 : dòng zú ) - yao ( 瑶族 : yáo zú ) - korean ( 朝鲜族 : cháoxiǎn zú ) - bai ( 白族 : bái zú ) - hani ( 哈尼族 : hāní zú ) - li ( 黎族 : lí zú ) , also called hlai in the ethnic group 's language . - kazak ( 哈萨克族 : hāsàkè zú ) - dai ( 傣族 : dǎi zú , also called dai lue , one of the tai ethnic groups ) - she ( 畲族 : shē zú ) - lisu ( 傈僳族 : lìsù zú )
