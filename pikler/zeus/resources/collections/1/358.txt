Home page     the home page of a web site is the document that a web server sends to another computer 's web browser application when it has been contacted without a request for specific information . that is , when you enter only a domain name in the address box without specifying a directory or a file , the `` home page '' is usually the first part of the web site you will be taken to . the home page is also called the main page . a properly written home page will tell a user about the information available in the web site , and how to view different parts of the web site . the home page of `` simple.wikipedia.org '' can be found at this link . home page was a popular computer application used for composing web pages . in linux-based servers , the homepage usually is index.html , index.php , index.cgi , etc . unless the website administrator has created a redirect to a specific webpage . however , in windows-based servers , the homepage is default.html , default.php , etc . this is a problem for website administrators to install website applications like mediawiki . mainly because most website applications are created with the homepage as index.php ( for php applications ) .
