Yes (band)    yes ( band ) yes are a progressive rock band that formed in london , england in 1968 that is among the most influential progressive bands of all time . the band has received wide critical praise for their work . their best known songs include `` owner of a lonely heart '' , `` roundabout '' and `` i 've seen all good people '' . discography . - studio albums - `` yes '' ( 1969 ) - `` time and a word '' ( 1970 ) - `` the yes album '' ( 1971 ) - `` fragile '' ( 1971 ) - `` close to the edge '' ( 1972 ) - `` tales from topographic oceans '' ( 1973 ) - `` relayer '' ( 1974 ) - `` going for the one '' ( 1977 ) - `` tormato '' ( 1978 ) - `` drama '' ( 1980 ) - `` 90125 '' ( 1983 ) - `` big generator '' ( 1987 ) - `` union '' ( 1991 ) - `` talk '' ( 1994 ) - `` keys to ascension '' ( 1996 ) - `` keys to ascension 2 '' ( 1997 ) - `` open your eyes '' ( 1997 ) - `` the ladder '' ( 1999 ) - `` magnification '' ( 2001 ) - `` fly from here '' ( 2011 ) - `` heaven & earth '' ( 2014 )
