Virtual private server     a virtual private server ( vps ) is a virtual machine ( something like a computer divided in to many smaller machines using special software ) . they are often used by internet hosting providers . multiple vps runs on the same machine , but they are separated as they will only share hardware resources without interfering on software level . a vps is dedicated to the customer 's needs , has the privacy of a separate physical computer , and can run server software . there are many platforms that can run vps . some of them are vmware and xen .
