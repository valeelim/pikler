GNU Affero General Public License     the gnu affero general public license is a copyleft license for software that is intended to be run on web servers like web applications . like the gnu general public license , the affero license allows you to use , copy and change the code of software , but requires that you release your changes if you distribute the software or you run a modified copy on a web server on the internet . the affero license is intended to make it harder for people to avoid releasing modifications to free software by running the software as a web service ( see software as a service ) . related pages . - list of software licenses - gnu general public license - free software foundation other websites . - why the affero gpl
