BioWare     bioware is a canadian video game developer company that was started in 1995. the headquarters for bioware is in edmonton , alberta . it was started by medical doctors ray muzyka , greg zeschuk , and augustine yip . the company currently has six studios : bioware edmonton , bioware montreal , bioware mythic , bioware austin , bioware san francisco , and bioware ireland . they were acquired by the american company electronic arts in 2008. they developed their first game `` shattered steel '' in 1996. the company is probably best known for the `` mass effect '' and `` dragon age '' video game series . games developed ( made ) . - `` shattered steel '' ( 1998 ) - `` baldur 's gate '' ( 1998 ) - `` '' ( 1999 ) - `` mdk2 '' ( 2000 ) - `` ( 2000 ) - `` ( 2001 ) - `` neverwinter nights '' ( 2002 ) - `` ( 2003 ) - `` ( 2003 ) - `` '' ( 2003 ) - `` jade empire '' ( 2005 ) - `` mass effect '' ( 2007 ) - `` '' ( 2008 ) - `` mass effect galaxy '' ( 2009 ) - `` '' ( 2009 ) - `` mass effect 2 '' ( 2010 ) - `` '' ( 2010 ) - `` dragon age ii '' ( 2011 ) - `` '' ( 2011 ) - `` mass effect 3 '' ( 2012 ) games in development . - `` '' ( 2014 )
