List of Wikipedias     this is a list of many of the different language editions of wikipedia ; as of august 2012 , there are 285 wikipedias . for their number of articles , see the . each wikipedia has a code , which is used as a subdomain below wikipedia.org . interlanguage links are sorted by that code . the codes represent the language codes defined by iso 639-1 and iso 639-3 , and the decision of which language code to use is usually determined by the ietf language tag policy . one code is not a language code ( 'be-x-old ' ) but refers to a specific orthography . some deviations include : size is given in decadic logarithm of number of articles . '6 ' means more than 1 000 000 , '5 ' more than 100 000 , '4 ' more than 10 000 .
