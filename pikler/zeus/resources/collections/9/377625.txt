Opel Signum     the opel signum is a automobile produced by opel . it was a more upmarket version of the vectra that it was based on and it arrived in 2003 with a facelift in 2005 and was discontinued in 2008 with no replacement . it was sold in the uk as the vauxhall signum . it is a hatchback version of the opel vectra . engines . - `` 1.8 '' 90 kw ( 122 hp ) and 103 kw ( 140 hp ) - `` 2.0 turbo '' 129 kw ( 175 hp ) - `` 2.2 direct '' 114 kw ( 155 hp ) - `` 2.8 turbo '' 169 kw ( 230 hp ) and 184 kw ( 250 hp ) - `` 3.2 '' 155 kw ( 211 hp ) - `` 1.9 cdti '' 74 kw ( 100 hp ) , 88 kw ( 120 hp ) and 110 kw ( 150 hp ) - `` 2.0 dti '' 74 kw ( 100 hp ) - `` 2.2 dti '' 92 kw ( 125 hp ) - `` 3.0 cdti '' 130 kw ( 177 hp ) and 135 kw ( 184 hp )
