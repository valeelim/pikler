List of rivers of Bangladesh     this is a list of rivers in bangladesh , by letter , with some bangladeshi names given for some of the rivers ; there are at least 31 known rivers of bangladesh : - arial kha river - atrai river - bangali river ( বাঙালি `` bangali '' ) - baral river ( বড়াল `` bôṛal '' ) - biskhali river - bhoirov river - brahmaputra river ( ব্রহ্মপুত্র `` bromhoputro '' ) - buriganga river ( বুড়িগঙ্গা `` buṛigôngga '' ) - bura gauranga river - dhaleshwari river ( ধলেশ্বরী `` dhôleshshori '' ) - dhepa river - feni river ( ফেনী feni ) - gorai-madhumati river ( গড়াই-মধুমতি `` gôṛai-modhumoti '' ) - gumti river - haora river - jaldhaka river ( জলঢাকা `` jôlḍhaka '' ) - jamuna river ( যমুনা `` jomuna '' ) - karnaphuli river ( কর্ণফুলি `` kôrnofuli '' ) - khowai river - kushiyara river - karotoa river - mahananda river ( মহানন্দা `` môhanônda '' ) - manu river , tripura - meghna river ( মেঘনা `` meghna '' ) - muhuri river ( মুহুরী `` muhuri '' ) - naf river ( নাফ `` naf '' ) - padma river ( পদ্মা `` pôdda '' ) - punarbhaba river - pusur river - shitalakshya river ( শীতালক্ষ্যা `` shitalokkha '' )
