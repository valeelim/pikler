BeatrIX     beatrix is a gnu/linux , live cd-based linux distribution started by steven watsky , it was designed to look very simple , in order to be as easy to use as possible to new users . a new beatrix desktop has just four icons ( e-mail , instant messaging , word processing and web browser ) . hence , anyone who has some computer experience will be able to use beatrix . beatrix is a small operating system , taking up only 200mb . the livecd fits into 80 mm discs that can store up to 184 mb of data . beatrix uses linux 2.6.7 and gnome 2.8 as its gui desktop . it has productivity tools such as openoffice.org 1.1.2 , mozilla firefox ( version 1.0 ) web browser , gaim ( version 1.0 ) , etc . beatrix has been discontinued since 2005. another distro called beafanatix is using the code of beatrix and continuing development . other pages . - gnu/linux - list of linux distributions other websites . - softpedia beatrix linux - beafanatix home page - download beatrix
