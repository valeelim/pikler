Assamese language     assamese ( { { lang-as|অসমীয়া } } translated as '' ôxômiya '' ) ( { { ipa2|ɔxɔmija } } ) is the easternmost indo-aryan language , spoken most in the state of assam in north-east india . it is also the main language of assam . it is spoken in parts of arunachal pradesh and other northeast indian states .
