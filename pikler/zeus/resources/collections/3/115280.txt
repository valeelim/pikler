Webmin     webmin is a web-based system configuration tool for opensolaris , linux and other unix-like systems . it can configure many operating system internals , such as users , disk quotas , services , configuration files , as well as modify and control many opensource applications , such as the apache http server , php and mysql . webmin is largely based on perl , running as its own process and web server . it defaults to tcp port 10000 for communicating , and can be configured to use ssl if openssl is installed with extra required perl modules .
