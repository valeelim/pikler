Rosalía de Castro     rosalía castro de murguía better known as rosalía de castro ( 24 february 1837 – 15 july 1885 ) was a galician writer and poet . a native of santiago de compostela in the galicia region of northwest spain , she wrote in both galician and castilian . poetry - la flor ( 1857 ) ( `` the flower '' ) - a mi madre ( 1863 ) ( `` to my mother '' ) - en las orillas del sar ( 1884 ) prose - la hija del mar ( 1859 ) ( `` the daughter of the sea '' ) - flavio ( 1861 ) - el cadiceño ( 1863 ) - ruinas ( 1866 ) - las literatas ( 1866 ) - el caballero de las botas azules ( 1867 ) - el primer loco ( 1881 ) - el domingo de ramos ( 1881 ) - padrón y las inundaciones ( 1881 )
