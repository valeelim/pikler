ZIP (file format)    zip ( file format ) the zip file format is a popular lossless data compression and archival format created by phil katz , founder of pkware . a zip file contains one or more files that have been compressed , to reduce their file size , or stored as-is . a number of compression algorithms are permitted in zip files but in 2008 only deflate is widely used and supported . zip files generally use the file extensions `` .zip '' or `` .zip '' and the mime media type codice_1 . some software uses the zip file format as a wrapper for a large number of small items in a specific structure . generally when this is done a different file extension is used .
