Prophets of Islam     prophets of islam are human beings who are regarded by muslims to be prophets chosen by god . all prophets preached the same message . there are 25 prophets mentioned in the qur'an . these are : 1. adam ( آدم ) 1. idris ( إدريس ) 1. nuh ( نوح ) ( noah ) 1. saleh ( صالح ) 1. hud ( هود ) 1. ibrahim ( إبراهيم ) ( abraham ) 1. lut ( لوط ) ( lot ) 1. ismail ( إسماعيل ) ( ismael ) 1. ishaq ( إسحاق ) ( isaac ) 1. ya'qub ( يعقوب ) ( jacob ) 1. yusuf ( يوسف ) ( joseph ) 1. ayub ( أيوب ) ( job ) 1. shu'aib ( شعيب ) 1. musa ( موسى ) ( moses ) 1. harun ( هارون ) ( aaron ) 1. dul-kifl ( ذو الكفل ) 1. dawud ( داود ) ( david ) 1. suleyman ( سليمان ) ( solomon ) 1. ilyas ( إلياس ) ( elijah ) 1. al-yasa ( اليسع ) ( elisha ) 1. yunus ( يونس ) ( jonah ) 1. zakaria ( زكريا ) ( zechariah ) 1. yahya ( يحيى ) ( john the baptist ) 1. isa ( عيسى ) ( jesus ) 1. muhammad ( محمد ) muslims believe in other prophets other than those mentioned by name in the qur'an as there are many verses in the qur'an which speak about it . these other prophets include : joshua , luqman , caleb , seth , daniel , isaiah , hosea , jeremiah , shem , benjamin , khidr , dhul qurnayn .
