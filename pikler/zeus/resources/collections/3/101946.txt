WAV     wav ( or wave ) , short for waveform audio file format , is a microsoft and ibm audio file format standard for storing an audio bitstream on pcs . it resembles the riff bitstream format in the sense that it stores data in “ chunks ” . it is also close to the iff and the aiff format used on amiga and macintosh computers , respectively . it is the main format used on windows systems for raw and typically uncompressed audio . the default bitstream encoding is the microsoft pulse code modulation ( lpcm ) format .
