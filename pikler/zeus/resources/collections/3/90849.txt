Death Cab for Cutie     death cab for cutie ( death cab or dcfc ) is an american indie band . the band was started in 1997 in bellingham , washington . the band members are ben gibbard ( singer , guitar ) , chris walla ( guitar , production ) , nick harmer ( bass ) and jason mcgerr ( drums ) . the band 's name was taken from the song `` death cab for cutie '' , written by neil innes and vivian stanshall and performed by their band the bonzo dog doo-dah band in the beatles ' 1967 film `` magical mystery tour '' . members . - ben gibbard - vocals , guitar , piano ( 1997-present ) - chris walla - guitar , production , piano ( 1997-present ) - nick harmer - bass ( 1997-present ) - jason mcgerr - drums ( 2003-present ) former - nathan good - drums ( 1997-1999 ) - michael schorr - drums ( 2000-2003 ) discography . studio albums . - `` something about airplanes '' ( 1998 ) - `` we have the facts and we 're voting yes '' ( 2000 ) - `` the photo album '' ( 2001 ) - `` transatlanticism '' ( 2003 ) - `` plans '' ( 2005 ) - `` narrow stairs '' ( 2008 ) - `` codes and keys '' ( 2011 ) - `` tba '' ( 2014 )
