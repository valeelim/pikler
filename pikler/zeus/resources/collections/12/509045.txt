La Wally     la wally is an italian opera in four acts by alfredo catalani . it was first performed in milan on march 28 , 1896. the libretto was written by luigi illica . `` la wally '' was a great success . principal roles . - wally - soprano - giuseppe hagenbach - tenor - vincenzo gellner - baritone - stromminger - bass discography . - 1929 : giannina arangi-lombardi ( wally ) , aureliano pertile ( hagenbach ) , carlo galeffi ( gellner ) , nazzareno de angelis ( stromminger ) ; coro e orchestra del teatro alla scala di milano , lorenzo molajoli - ( arkadia ) - 1965 : antonietta stella ( wally ) , franco corelli ( hagenbach ) , mario sereni ( gellner ) , bonaldo giaiotti ( stromminger ) ; coro e orchestra del teatro reale dell'opera di roma , gabriele santini - ( emi classics ) - 1968 : renata tebaldi ( wally ) , mario del monaco ( hagenbach ) , piero cappuccilli ( gellner ) , justino díaz ( stromminger ) ; coro lirico di torino , orchestre national de l'opéra de monte carlo , fausto cleva - ( decca ) - 1971 : montserrat caballé ( wally ) , luciano pavarotti ( hagenbach ) , rolando panerai ( gellner ) , nicolai ghiaurov ( stromminger ) ; ambrosian opera chorus , london philharmonic orchestra , georg solti - ( decca )
