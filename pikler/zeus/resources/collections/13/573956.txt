ActiveX     activex is a control in the `` dll '' ( dynamic-link library ) that can be used as a share data over the applications . activex is a type of library in the file dll . any of the microsoft windows data files having the same file format as that of the dynamic-link library can be called a resource dll . examples of such dlls include icon libraries , sometimes having the extension icl , and the font files , having the extensions fon and fot .
