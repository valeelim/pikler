Form-based interface     a form-based interface is a kind of user interface . in it , the user interacts with the application by selecting one of a number of possible values , and by entering text into the fields that accept it . a word processor which is used to write documents , might offer settings for the font size , the font to use , and the alignment of the paragraph on the page . many databases support a technology called query by example : users who do not know sql can easily select database records which are similar to the information entered .
