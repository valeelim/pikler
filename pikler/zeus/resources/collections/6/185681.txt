Daemon (computer software)    daemon ( computer software ) in unix , and in other computer multitasking operating systems , a daemon ( or ) is a computer program that runs as a background task . many daemons have names that end with the letter `` d '' : for example , syslogd , the daemon that handles the system log , httpd , a daemon that handles web requests , or sshd , which takes care of incoming ssh connections . related pages . - server ( computing ) other websites . - start-stop-daemon man page ( debian package ) - tutorial : unix daemons in perl - how to daemonize in linux - unix daemon server programming - http daemon in bash
