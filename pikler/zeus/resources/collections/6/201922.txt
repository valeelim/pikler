Portable application creators     portable application creators allow the creation of portable applications ( also called : portable apps ) . no `` agent '' / '' client '' is required for these ( also called `` agentless '' solutions ) : - vmware thinapp - xenocode virtual application studio isv edition - installfree - ceedo ( using `` argo application installer '' ) - landesk application virtualization - cde ( linux ) these programs require an agent or `` client '' ( to run the `` compiled '' app ) - citrix xenapp - microsoft application virtualization - symantec endpoint virtualization suite - appzero - endeavors application jukebox
