LinuxLive USB Creator     linuxlive usb creator is a free software for windows that allows users to create bootable live usb systems for many linux distributions . this software is designed to be as easy to use as possible . its graphical interface uses a step by step approach and some traffic lights indicates to the user if each mandatory steps are validated . features . - creates bootable live usb of many linux distributions - makes persistent installs to save all documents created and modifications made to the system - run linux directly in windows using portable virtualbox - hides created files in windows - non-destructive install ( does not format the device ) supported linux distributions and variants . - ubuntu / kubuntu / xubuntu - centos - debian live gnome/kde/lxde/xfce - damn small linux - puppy linux - toutou linux - gparted live - clonezilla - fedora 11 kde/gnome ) - kuki - linux mint main edition - crunchbang standard & lite editions - super os other pages . - linuxlive usb creator - official website
