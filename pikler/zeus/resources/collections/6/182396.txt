Amorgos     amorgos is an island in the aegean sea . it is a part of greece and a part of the cyclades . its area is 126,3 km . there is one municipality which has 1859 inhabitants . its highest mountain krikellos has 821 m. the largest towns and settlements are : - aigiali ( pop . 229 ) - chora ( amorgos ) ( 398 ) - arkesini ( 129 ) - katapola ( 128 ) - ormos egialis ( 125 ) - potamos ( 116 ) - rachidi ( 148 ) - tholaria ( 147 ) - xylokeratidion ( 107 ) other websites . - official website
