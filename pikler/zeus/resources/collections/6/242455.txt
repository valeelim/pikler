Game Maker     game maker is a computer program for the windows operating system . it was first made by mark overmars , but it is now made by the company yoyo games . game maker lets people make computer games without knowing a computer programming language . people who know how to use programming languages can use game maker 's own programming language , called game maker language . the newest version is game maker studio . since the first version made in 1999 , game maker was given many new features . one of these features is 3-d graphics . game maker has a drag-and-drop interface . this lets people make games using mostly the computer mouse . game maker uses icons that tell the person what actions they can do . people can make simple games by moving these icons into a given order . more detailed games can be made when game maker language is used . game maker has an internet community and website . there , people who use game maker can put their games up for others to play . people can also get help on using game maker from other people .
