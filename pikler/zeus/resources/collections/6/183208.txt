Browser game     a browser game is a video game that is played on a web browser . it is different from other video and computer games in that it normally does not need any software to be installed on the user 's computer , except for the web browser . there are games that rely solely on client-side technologies such as javascript or common plugins such as java or flash player , whereas some employ server-side scripting . the latter case consists typically of ( massive ) multiplayer games , whereas the client-side games are typically single-player games . a game played in a browser is often called a browser-based game .
