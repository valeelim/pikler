Telephone numbers in Belarus     country code : +375 < br > international call prefix : 00 ( was 8~10 ) < br > trunk prefix : 8 belarus began using its own country code +375 in 1995. before that it was part of the +7 international country code . the local numbering plan came from ussr . it has changed little . geographic area codes were made based on previous 3-digit ussr area codes by removing the lead `` 0 '' . calling across areas still requires dialing trunk prefix 8 and waiting for dialtone after that .
