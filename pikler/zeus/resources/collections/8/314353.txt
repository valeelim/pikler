Multivalued function     a multivalued function ( multifunction ) is a relation like a function . a multifunction associates each input with `` at least one '' output . a strict function associates each input with `` exactly one '' output . an example of a multivalued function is the square root . every complex number ( except for 0 ) , has two square roots . example : formula_1 could be 2 or -2 because formula_2 and formula_3 .
