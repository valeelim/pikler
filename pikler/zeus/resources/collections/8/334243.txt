List of counties in New Hampshire     there are 10 counties in the u.s. state of new hampshire . five of the counties were formed in 1769. this was when new hampshire was still an english colony and not a state . the fips county code is the five-digit federal information processing standard ( fips ) code which uniquely identifies counties and county equivalents in the united states . the three-digit number is unique to each individual county within a state . the code for new hampshire is 33. the links in the column `` fips county code '' are to the census bureau info page for that county .
