British Rail Class 403     the southern railway ( sr ) gave the designation 5bel to the 5-car all-pullman electric multiple units built to work the prestigious brighton belle trains between london victoria and brighton . these units survived long enough in british rail ownership to be allocated tops class 403. between 1933 and 1935 the units were designated 5pul ( the 'pul ' code was then used for the 6pul units ) .
