Windows Product Activation     windows product activation is a service built into all versions of windows from windows xp and windows server 2003 above to stop people downloading pirated copies . the user has 30 days to activate windows and after 30 days , the computer will not run properly until it is activated . when windows is first installed , there will be a little icon on the screen with a message asking for windows to be activated . then an activation wizard will appear and the user has 2 options on activating windows . activation could be done on the internet , or on the telephone . the quickest way to activate is by the internet , but if that fails , then the user would need to call microsoft for the activation id . the activation id would be read by the representative and then the user would type in the key on the activation wizard screen .
